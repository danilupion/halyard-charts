Thank you for installing {{ .Chart.Name }}!

Your Obsidian instance is starting up.

1. Access the web UI inside the cluster:

  kubectl port-forward --namespace {{ .Release.Namespace }} svc/{{ include "common.fullname" . }} 8080:{{ .Values.service.port }}

  Then open http://localhost:8080 in your browser.

2. Configure persistence:

{{- if not .Values.persistence.config.enabled }}
  WARNING: persistence.config.enabled=false. Vault changes will be lost if the pod is recreated.
  Enable persistence to store your vaults:
    helm upgrade {{ .Release.Name }} ./ --set persistence.config.enabled=true
{{- else if not .Values.persistence.config.existingClaim }}
  A PersistentVolumeClaim named {{ include "common.fullname" . }}-config will be created.
{{- else }}
  Using existing claim {{ .Values.persistence.config.existingClaim }} for Obsidian vaults.
{{- end }}
